About

About this project

Built as part of an ICT course to practice working with open geospatial and humanitarian datasets. It takes the Dartmouth Flood Observatory’s global archive and turns it into interactive browser-based tools.

Everything runs client-side — no server, no database, no installation. Open the HTML and the data loads instantly.

The map viewer supports drag-and-drop CSV upload, so you can visualise any dataset with latitude and longitude columns.

View on GitHub
  • Dartmouth Flood ObservatoryGlobal flood archive — 5,130 events (1985–2021) with location, severity, cause, and displacement data.
  • Chart.jsOpen-source charting library for the analytics dashboard.
  • Leaflet.jsLightweight mapping library for the interactive flood viewers.
  • PapaParseBrowser-native CSV parser — loads the dataset without a server.
