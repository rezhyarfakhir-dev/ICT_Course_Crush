About

About this project

This project was built as part of an ICT course to practice working with open geospatial and humanitarian datasets. It takes the Dartmouth Flood Observatory's global flood archive and turns it into two interactive browser-based tools.

Everything runs locally in the browser — no server, no database, no installation needed. Just open the HTML file and the data loads instantly.

The map viewer also supports drag-and-drop CSV upload, so you can visualise any dataset that contains latitude and longitude columns.

View on GitHub
  • Dartmouth Flood Observatory (DFO) Global flood archive with 5,130 events from 1985–2021, including location, severity, cause, and displaced populations.
  • Chart.js Open-source JavaScript charting library used for the analytics dashboard.
  • Leaflet.js Lightweight open-source mapping library used for the interactive flood map viewer.
  • PapaParse Fast, browser-native CSV parser used to load and stream the dataset without a server.
